# 题库建设执行计划：高项近3年真题

> 目标：在W3开发启动前完成题库数据录入

## 一、录入范围

### 综合知识真题
- 2023-2025年高项上午真题
- 每年约75题×3年≈225题（目标400题含模拟题）
- 按十大知识领域+ITTO分类

### 论文题目
- 2019-2025年高项论文真题约20题

### 案例分析
- 2021-2025年高项下午真题约15套

## 二、数据格式

综合知识JSON Schema含9个字段：id, subject, year, knowledge_area, process_group, difficulty, type, content, options, correct_answer, basic_explanation, tags

## 三、标签体系

- 十大知识领域标签（integration_management~stakeholder_management）
- 五大过程组标签（initiating~closing）
- ITTO标签（itto:{过程名}）
- 难度标签（1-5星）

## 四、质量验收标准

- 题目准确性≥98%
- 每题≥3个标签
- 解析质量：要点提示≥2条
- JSON Schema校验100%通过
