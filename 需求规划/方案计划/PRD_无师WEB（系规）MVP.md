# PRD：无师WEB（系统规划与管理师）MVP

> 版本：V1.0 | 状态：规划完成

## 一、产品概述

唯一专注软考系统规划与管理师论文写作与AI批改的Web工具。

目标用户：备考系规的IT从业者，年龄25-40岁，日均PC使用8小时+

北极星指标：Web端完成一篇完整论文并被AI批改的用户数/周

## 二、用户故事

### Epic 1：论文写作（P0）

**US-001：三栏论文编辑器**
三栏同屏显示题目大纲、写作区、AI建议，面板可折叠，支持Ctrl+S

**US-002：自动保存**
每30秒本地保存，每5分钟云端同步，关闭前提示

**US-003：字数统计**
实时显示字数，接近2500字黄色预警，超过红色提示

### Epic 2：AI批改（P0）核心

**US-004：逐段AI批改**
SSE流式返回逐段批注，每段含问题描述、严重程度、修改建议
原文问题区域红/黄/绿高亮，支持一键应用建议

**US-005：多维度评分**
4维度：论点、论证、语言、结构，总分百分制

**US-006：批改历史与对比**
查看历次批改，相邻两次分数对比，进步曲线

### Epic 3：素材库（P0）

**US-007：素材管理**
支持论点/案例/模板三种类型，标签分类，拖拽到写作区

### Epic 4：个人主页（P0）

**US-008：Dashboard**
总论文数/平均分数趋势、写作进步曲线、本周学习时长、考试倒计时

## 三、非功能需求

- 性能：首屏<2s，编辑器延迟<50ms
- 兼容：Desktop Chrome/Edge/Firefox
- 安全：JWT + HTTPS + XSS防护
- 数据：自动备份 + 导出Markdown/PDF
