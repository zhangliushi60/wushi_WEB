# MVP范围与Sprint排期 — 无师WEB

> 8周交付：论文核心版（写作+批改+素材库）

## 一、MVP范围

纳入MVP（P0）：
- 用户系统：注册/登录/JWT
- 论文写作：三栏Markdown编辑器 + 自动保存
- 论文批改：AI逐段批改（SSE流式）+ 评分 + 维度拆解
- 论文管理：列表/详情/删除/版本管理
- 素材库：论点/案例/模板 CRUD
- Dashboard：基础统计 + 进步曲线

排除MVP（P1/P2）：
- 题库练习、案例分析、模拟考试、查重、小程序/APP

## 二、Sprint排期（8周）

### Sprint 0（W1）：基础设施
- 项目脚手架、数据库设计、用户认证、前端路由+布局、UI组件库

### Sprint 1（W2-3）：论文写作核心
- Tiptap编辑器集成、三栏布局、自动保存、论文CRUD API、字数统计

### Sprint 2（W4-5）：AI批改
- AI Gateway路由层、DeepSeek V3集成、SSE流式批改、逐段批注高亮渲染

### Sprint 3（W6-7）：素材库 + Dashboard
- 素材库CRUD、拖拽素材、Dashboard统计API、进步曲线

### Sprint 4（W8）：打磨 + 部署
- 错误处理、响应式、深色模式、安全加固、部署上线、内测

## 三、里程碑

- W1: 基础设施就绪
- W3: 论文写作可用
- W5: AI批改上线
- W7: 完整闭环
- W8: 上线

## 四、风险

| 风险 | 应对 |
|------|------|
| Tiptap定制超预期 | 降级为简易Textarea+预览 |
| DeepSeek API不稳定 | GPT-4o/Claude备用 |
| 批改Prompt效果不佳 | 预留1周调优 |
| 人力不足 | 砍素材库，优先写作+批改 |
