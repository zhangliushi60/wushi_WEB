# PRD：软考AI私教产品 — MVP（软件设计师方向）

> 文档版本：V1.1 | 撰写日期：2026年7月4日 | 修订日期：2026年7月9日
> MVP范围：软件设计师（中级）单科 + AI深度解析 + 智能推荐刷题 + 千人千面AI备考计划
> 预期上线：MVP 0.5（8周）→ MVP 1.0（16周）

## 一、背景与目标

软考是IT从业者职称评定、积分落户、职业晋升的核心凭证。2024年报考107.22万人次，2026年突破342万人次，86.7%为在职技术人员。

核心痛点：中级通过率仅15%-25%，AI渗透率极低，在职人员备考时间碎片化，解析质量差，考纲更新频繁。

产品目标：AI深度解析满意度≥85%，付费转化率≥3%，日均答题行为≥5000次。

## 二、MVP范围定义

| 功能模块 | MVP 0.5（0-8周） | MVP 1.0（8-16周） |
|----------|:----------------:|:-----------------:|
| 千人千面AI备考计划 | 备考偏好采集+AI每日计划+任务打卡 | +反馈自适应调整 |
| AI深度解析 | 追问式解析+错误归因 | +多解法对比 |
| 基础题库 | 软件设计师近3年真题 | +章节练习扩充 |
| 错题本 | 自动归集+考点标注 | +定期回顾提醒 |
| 用户诊断 | 初始水平测试 | +知识点掌握度评估 |
| 智能推荐刷题 | — | 基于错题关联推荐+难度梯度 |
| 每日一练 | 每日推送10题 | +个性化推送 |

## 三、用户故事（8个核心US）

- US-01：在职开发者碎片化刷题
- US-02：AI追问式解析
- US-03：错题智能归集与回顾
- US-04：初始水平诊断
- US-05：智能推荐刷题（MVP 1.0）
- US-06：免费用户价值感知
- US-07：千人千面AI备考计划
- US-08：计划自适应调整

## 四、功能架构

无师APP包含模块：千人千面AI备考计划（4子模块）、题库模块（4子模块）、AI解析模块（3子模块）、刷题模块（4子模块）、错题本模块（3子模块）、诊断模块（2子模块）、用户中心（4子模块）

## 五、核心流程

新用户路径：注册登录→3步备考引导→AI计划生成→初始水平测试→首页计划卡片→刷题→解析→错题本→会员转化

## 六、页面清单（15页）

启动页、注册登录页、备考引导页、首页（含计划卡片）、章节列表页、刷题页、解析页、AI追问对话页、错题本页、知识点雷达图页、每日一练页、模拟考试页(1.0)、智能推荐页(1.0)、会员购买页、个人中心

## 七、数据指标

北极星指标：周活跃学习天数（WAU Learning Days）
MVP目标：注册5000+，DAU/MAU≥20%，次日留存≥30%，AI解析满意度≥85%

埋点方案：25个核心事件（app_launch到membership_trial_expire）

## 八、验收标准

功能验收10项（引导完成率90%+、AI解析准确率95%+），性能验收5项（启动≤2s、AI解析≤5s），内容质量验收4项（答案100%准确、AI解析≥95%准确）

## 九、边界场景与异常处理

11个边界场景（退出续答、断网缓存、题目不足、AI追问上限等），7种异常处理策略（API超时降级、支付失败重试、未登录跳转等）

## 十、权限规则

用户权限矩阵：12项功能×3级用户（未注册/免费/会员），会员定价¥39/月，含3天试用。
