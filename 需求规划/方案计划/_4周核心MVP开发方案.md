# 无师APP 软考AI私教 — 4周核心MVP开发方案

> 版本: V1.0（4周精简版）| 日期: 2026年7月7日
> 原计划: 16周5个Sprint → 压缩至4周核心功能上线

## 一、产品概述

面向信息系统项目管理师（高级）备考者的移动端AI辅助学习应用，4周MVP聚焦核心刷题闭环：
注册登录 → 综合知识章节刷题 → AI深度解析 → 错题归集回顾 → 会员付费转化

### 范围裁剪决策

保留：用户系统、综合知识刷题闭环、AI深度解析(简化单轮)、错题本(基础版)、会员支付(基础版)、首页
砍掉延后：AI论文批改、AI案例分析、AI追问对话、诊断雷达图、每日一练、推送通知、富文本编辑器、查重检测、智能推荐算法、CI/CD流水线、微信OAuth登录

## 二、4周迭代总览

| 周 | 核心任务 | 里程碑 |
|:--:|----------|--------|
| W1 | RN脚手架 + 后端骨架 + 登录API + 组件库 + 题库导入 | 基础设施就绪 |
| W2 | 章节列表/题目/判分API + 刷题页 + 基础解析页 | 刷题闭环可用 |
| W3 | AI解析Prompt/DeepSeek接入/SSE流式输出 + 错题本 + 首页 | AI功能可体验 |
| W4 | 微信/支付宝支付 + 权限中间件 + 会员页 + 全链路联调 + 打包发布 | MVP正式上线 |

## 三、技术架构

- 前端：React Native + TypeScript
- 后端：Node.js + Express + Prisma
- 数据库：MySQL 8.0 + Redis 7.x
- AI：DeepSeek V3（主）/ GPT-4o-mini（备选）
- 支付：微信支付v3 + 支付宝

## 四、关键验收标准

- 刷题页面加载时间 ≤1秒
- AI解析响应时间 ≤5秒
- AI解析准确率 ≥95%
- 支付成功率 ≥95%
- API可用性 ≥99%
