# 需求规划_无师APP_V2_软考

## 一、V2版本定位

基于V1的市场反馈，聚焦软考，深化AI能力

## 二、软考考试体系

软考分为：
- 初级：程序员、网络管理员等
- 中级：软件设计师、网络工程师、系统集成项目管理工程师等
- 高级：信息系统项目管理师（高项）、系统规划与管理师（系规）、系统架构设计师等

## 三、V2核心功能

### 3.1 题库建设
- 历年真题（近5年）
- 模拟题
- 按章节/知识点/难度分类

### 3.2 AI功能升级
- 主观题AI评分
- 论文AI批改
- 案例分析AI辅助

### 3.3 学习体系
- 知识图谱
- 错题分析
- 薄弱点诊断

## 四、优先支持的考试科目

Phase 1：信息系统项目管理师（高项）- 考生最多
Phase 2：系统规划与管理师（系规）
Phase 3：软件设计师

## 五、里程碑

- M1(3个月)：高项题库+基础功能
- M2(6个月)：AI批改+系规
- M3(9个月)：全科目覆盖
