# Web端功能架构与页面设计规格

## 一、信息架构

- 首页Dashboard
  - 论文写作（核心）：新建论文、我的论文列表、论文编辑器（三栏）
  - 论文批改：批改历史、批改详情页
  - 案例分析：真题列表、答题页（双栏）
  - 题库练习：章节练习、模拟考试、错题本
  - 素材库：论点收藏、案例库、模板管理
  - 个人中心：学习统计、写作进步曲线、账号设置

## 二、论文编辑器（三栏布局）核心页面

左侧面板(280px): 题目、大纲、素材库（可折叠）
中间写作区(自适应): 格式工具栏 + Markdown编辑器
右侧面板(320px): AI助手、实时统计、智能提示、评分标准

## 三、批改详情页

左侧论文原文（带批注高亮），右侧AI批改面板
- 总体评分 + 4维度评分
- 逐段建议（点击段落查看）
- 与上次对比分数

## 四、全局设计规范

配色：主色 #2563EB 蓝色，强调色 #F59E0B 琥珀色
桌面优先：论文写作 Desktop Only（这不是BUG，是产品定位）
