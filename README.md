# 无师WEB — 软考系统规划与管理师备考平台

> 基于 AI 的 Web 端备考工具，专为系统规划与管理师（系规）案例分析 & 论文写作场景优化。

## 项目概述

原有无师 APP 在移动端案例分析和论文写作体验较差，本项目将核心写作场景迁移至 Web 端，提供：
- **三栏论文编辑器**：题目/写作/批注同屏
- **AI 多模型分流**：DeepSeek V3 + GPT-4o/Claude
- **SSE 流式输出**：实时 AI 反馈
- **逐段精批 + 查重**：论文批改核心能力

## 文档目录

- `需求规划/需求调研/` — 市场分析与需求调研
- `需求规划/方案计划/` — PRD、技术方案、MVP规划
- `需求规划/需求评审/` — 专家评审报告
- `需求规划/原型图/` — 高保真原型

## 技术栈

- 前端：React + TypeScript + Tiptap 编辑器
- 后端：Node.js + Express + Prisma
- AI：多模型分流（DeepSeek V3 / GPT-4o / Claude）
- 数据库：MySQL + Redis
